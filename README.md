# Innovation_project
A React framework for musician School works Non for profit

#how to use


git clone https://github.com/lucas-project/Innovation_project.git
cd Innovation_project
#Start frontend
cd client
npm i
npm start
#Start backend
cd server
npm i
npm start


#version control
node.js 16.0.0
react-router-dom v6
    use navigate instead of history
